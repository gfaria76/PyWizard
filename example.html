F-strings e Formatação

Apresentando dados reais com clareza: o mestre do código sabe que quem lê merece entender.

auto_awesome ARCANE GOAL

Dominar a arte da formatação para transformar números brutos em informações legíveis e profissionais.

A Retomada

Você já sabe invocar variáveis e coletar dados via input(). Mas como apresentar o resultado de uma poção sem que pareça apenas um erro de sistema?

"O código que não comunica é apenas ruído no vazio digital."
error

O Problema: Dados Crutos

Imagine um recibo de alquimista sem formatação:

1254.56789123

Quanto é isso? Dinheiro? Peso? Mana? Sem contexto, a informação é inútil.

menu_book Conceito: F-Strings (Formatted Strings)

SINTAXE BÁSICA
nome = "Mago" print(f"Olá, {nome}!")

Use o prefixo f e chaves {} para injetar variáveis.

CASAS DECIMAIS
v = 19.9912 print(f"Val: {v:.2f}")

O :.2f trava a exibição em exatamente 2 casas decimais.

PORCENTAGEM
taxa = 0.15 print(f"Taxa: {taxa:.1%}")

O :.1% multiplica por 100 e adiciona o símbolo %.

school Prática Guiada: O Alquimista Contador

GRIMOIRE OS v3.11
description main.py

# Sistema de Recibo Profissional

salario = float(input("Salário base: "))

bonus = salario * 0.10

total = salario + bonus

# Exibição com f-strings de alta precisão

print(f"--- HOLERITE MÁGICO ---")

print(f"Base: R$ {salario:,.2f}")

print(f"Bônus: R$ {bonus:,.2f}")

print("-----------------------")

print(f"TOTAL: R$ {total:,.2f}")
